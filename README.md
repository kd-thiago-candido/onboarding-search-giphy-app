# search-giphy-app

This app shows the 100 trending gifs currently according to [GIPHY](https://giphy.com/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
